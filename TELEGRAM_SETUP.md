# Настройка Telegram бота для формы

## Шаг 1: Создание бота

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Придумайте имя бота (например, `YandexSEO Pro Bot`)
4. Придумайте username бота (должен заканчиваться на `bot`, например `yandexseo_pro_bot`)
5. BotFather выдаст токен вида: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`

## Шаг 2: Получение Chat ID

1. Найдите своего бота в Telegram по username
2. Отправьте боту любое сообщение
3. Перейдите по ссылке: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
4. Найдите в ответе `"chat":{"id":123456789` — это ваш Chat ID

## Шаг 3: Настройка переменных окружения в Netlify

1. Зайдите в [Netlify Dashboard](https://app.netlify.com/)
2. Выберите ваш сайт
3. Перейдите в **Site settings** → **Environment variables**
4. Добавьте две переменные:
   - `TELEGRAM_BOT_TOKEN` = ваш токен от BotFather
   - `TELEGRAM_CHAT_ID` = ваш Chat ID

## Шаг 4: Пересоберите и redeploy

После добавления переменных окружения, Netlify автоматически пересоберёт сайт.

## Альтернативный способ (для тестирования)

Если вы хотите протестировать локально, создайте файл `.env` в корне проекта:

```
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_CHAT_ID=your_chat_id_here
```

**Важно:** Никогда не коммитьте `.env` файл в публичный репозиторий!

## Проверка работы

1. Заполните форму на странице Контакты
2. Нажмите "Отправить"
3. Проверьте Telegram — должно прийти уведомление с данными формы
